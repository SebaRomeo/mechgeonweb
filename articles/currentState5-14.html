State Update 5/14/23

New systems and mechanics added, some visual models and a name change: Mech-Geon: Tactics.

OverWatch mechanic

I took this action from Divinity, the idea is set your character in an alert state, if an enemy moves into your attack range your character will attack him automatically. In this case I set the corner of the board as the target location to watch but in the future this will depend on the character range

Node states

Each node is able to contain more than one state, for example: fire, electricity, water, etc... (these are examples, they may not be in the final game), if the player walk throught a node with a state set, the effects of that state will be applied to the node, if the state is fire, the player will receive fire damage. In this video the is no visual feedback but you can see that the player is destroyed for taking to much damage from the node.

Fog of war update

Now if the player tries to move to a node where there is another character but the player doesn't know it because that node is out of his vision range, his character will stop when the node is inside the fog of war.



These were the main updates so far. I also implemented a stats system for the actions, each action will consume an amount of some resource (stamina, energy, etc...). Those resources can be defined by the stats system but is in an early stage and because of that I didn't include it in the report.
